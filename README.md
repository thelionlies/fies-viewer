# CALABARZON FIES Viewer

A Django-based web application for exploring and analyzing household economic data from the **Family Income and Expenditure Survey (FIES)** in the CALABARZON Region, Philippines.

This project was developed for the Programming with Databases course in the MDSc program at Ateneo de Manila University.

---

## Project Overview

FIES Viewer allows users to:

- Visualize household income and expenditure patterns
- Perform CRUD operations on household records
- Filter and search data based on multiple criteria
- View aggregated provincial statistics with weighted averages

The application provides an interactive, user-friendly interface with responsive design and data visualizations.

---

## Dataset

The system uses the FIES dataset for CALABARZON households, sourced from the Philippine Statistics Authority. Data includes household demographics, income, and expenditure information.

---

## Technology Stack

- **Backend:** Django 5.2.8  
- **Database:** SQLite (development)  
- **Frontend:** Bootstrap 5.3.0  
- **Visualization:** Chart.js  
- **Python Version:** 3.14  

---

## Features

- Paginated household listings and detailed views
- Multi-criteria search and filtering
- Auto-calculated income, expenditure, and per capita values
- Weighted provincial statistics for accurate regional insights
- Clean and responsive interface for ease of use

---

## Purpose

This project demonstrates database-backed web application development, combining data management, interactive visualization, and analytical querying in a single platform for regional socioeconomic analysis.
